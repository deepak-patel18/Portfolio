# 👨‍💻 About This Project

This is my **Full Stack Developer Portfolio Website** built using **React, Node.js, Express, and MongoDB**.

The portfolio showcases my **projects, technical skills, and contact information** in a modern animated UI.

It also includes an **Admin Dashboard** where I can dynamically **add and delete projects** without modifying the code.

# ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/deepak-patel18/Portfolio.git
```

---

### 2️⃣ Install Backend Dependencies

```
cd backend
npm install
```

---

### 3️⃣ Create `.env`

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

---

### 4️⃣ Run Backend

```
npm run dev
```

or

```
node server.js
```

---

### 5️⃣ Install Frontend Dependencies

```
cd frontend
npm install
```

---

### 6️⃣ Run Frontend

```
npm run dev
```
